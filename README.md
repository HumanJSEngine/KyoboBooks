# KyoboBooks

교보문고 미니프로젝트 입니다,

## 🖥️ 프로젝트 소개

Axios를 이용한 데이터 연동, JSX 문법을 이용한 UI,UX 연습

교보문고 DB를 이용하여 미니 온라인서점을 구현하였습니다.
json-server --watch data/db.json --port 8000로 임시 api 구현
